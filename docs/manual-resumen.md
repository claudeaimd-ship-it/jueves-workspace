## Descripción

Manual de operación para máquina inyectora de plástico vertical modelo VL4-400XR2. 
Cubre desde instalación, operación segura, mantenimiento, solución de problemas 
hasta especificaciones técnicas detalladas.

## Contenido del Manual

1. **Capítulo I - Seguridad** — Advertencias de seguridad, requisitos de protección personal, señales de seguridad
2. **Capítulo II - Descripción de la Máquina** — Unidades de inyección, sistema hidráulico, panel de control
3. **Capítulo III - Instalación** — Diagrama de instalación, conexiones eléctricas, montaje, condiciones ambientales
4. **Capítulo IV - Garantía** — Instrucciones y requisitos de garantía
5. **Capítulo V - Puesta en Marcha** — Procedimientos de calibración y configuración inicial
6. **Capítulo VI - Operación** — Procedimientos de operación, parámetros de moldeo
7. **Capítulo VII - Mantenimiento** — Mantenimiento preventivo y correctivo
8. **Capítulo VIII - Solución de Problemas** — Problemas comunes de moldeo por inyección y soluciones
9. **Apéndices** — Diagramas eléctricos, hidráulicos, lista de componentes

## Notas Técnicas

- Máquina de cierre vertical tipo columna
- Sistema hidráulico con bomba y motor
- Controlador de temperatura para barril y boquilla
- Husillo (tornillo) para plastificación
- Tolva para alimentación de material
- Incluye advertencias sobre manipulación de componentes calientes (anillos de calefacción)
- Sistema de seguridad con puertas de seguridad y paro de motor
